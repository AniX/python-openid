Type ConsumerAssociation

object --+
         |
        ConsumerAssociation

This class represents a consumer's view of an association. In general, users of this library will never see instances of this object. The only exception is if you implement a custom OpenIDStore.

If you do implement such a store, it will need to store the values of the server_url, handle, secret, issued, and lifetime instance variables.
